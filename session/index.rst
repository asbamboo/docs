SESSION模块[Session]
=============================

SESSION模块的作用主要，管理自定义的session handler。规范统一asbamboo项目中，session相关的代码处理方式。

如何安装？
-------------------------

#. 通过 `composer`_ 安装::

    composer require asbamboo/session
    
#. 从 https://github.com/asbamboo/session 获取。

-------------------------
