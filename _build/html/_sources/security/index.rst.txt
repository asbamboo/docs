安全模块[security]
=============================

安全模块[security]用来管理用户的登录、登出、角色、权限等访问控制。

如何安装？
-------------------------

#. 通过 `composer`_ 安装::

    composer require asbamboo/security
    
#. 从 https://github.com/asbamboo/security 获取。
